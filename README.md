## 그라운드 룰

---

- 깃 브랜치 룰
  - be-main
    - be-develop
    - be-feat#(issueNum)
  - fe-main
    - fe-develop
    - fe-feat#(issueNum)
- 커밋 컨벤션

  ```jsx
  [init] 개발 환경 초기 세팅
  [feat] 새로운 기능 추가
  [design] UI를 위한 스타일링
  [fix] 버그 수정
  [docs] 문서 추가, 수정, 삭제
  [style] 코드 포맷팅(코드 순서 변경, 세미콜론 추가)
  [refactor] 코드 리팩터링
  [asset] 폰트, 이미지 추가
  [chore] 그 외 자잘한 수정
  ```

- 이슈 컨벤션

  - [FE] 기능 설명~
    - 체크 박스로 각 task 정리
  - [BE] 기능 설명~

    - 체크 박스로 각 task 정리

  - 템플릿에 따라 작성
  - 태그 및 마일스톤에 정리
